Tên đề tài: Quản lý cửa hàng sách
Mục tiêu: 
- Chung: Xây dựng phần mềm giúp quản lý cửa hàng sách dễ dàng hơn
- Cụ thể: 
	+) Xây dựng cơ sở dữ liệu về sách, nhà xuất bản, cửa hàng sách.
	+) Xây dựng phần mềm hỗ trợ nhân viên, người quản lý.
		- Nhân viên: 
			- Tìm kiếm: Chức năng này giúp dễ dàng tìm kiếm sách 
			- Lập hóa đơn: Chức năng này giúp lập hóa đơn nhanh chóng 
		- Người quản lý: 
			- Tìm kiếm: Chức năng này giúp dễ dàng tìm kiếm sách 
			- Lập hóa đơn: Chức năng này giúp lập hóa đơn nhanh chóng 
			- Quản trị: 
				+ Cập nhật, xóa, thêm, bổ sung các thông tin về sách. 
				+ Quản lý khách hàng, nhân viên
				+ Quản lý hóa đơn 
				+ Quản lý kho
Nội dung:
	+) Tìm hiểu công nghệ phát triển ứng dụng trên windows
		- Ngôn ngữ lập trình C#
		- Lập trình winform 
		- Cơ sở dữ liệu: SQL Server
		- Công cụ lập trình: Microsoft Visual Studio, SQL Server Management Studio
